# Chenmin Sun Website Skeleton

This is a lightweight static website starter for GitHub Pages.

## Structure

- `index.html` — homepage / about page
- `research.html` — research page
- `teaching.html` — teaching and academic service page
- `blog/index.html` — blog landing page
- `blog/welcome.html` — sample post
- `assets/css/style.css` — shared stylesheet
- `assets/js/main.js` — mobile navigation + active link highlighting
- `assets/files/` — place CV, PDFs, or downloadable files here
- `assets/images/` — place profile image or figure assets here

## First edits to make

1. Replace placeholder links (`#`) with real arXiv / ResearchGate / Google Scholar / journal links.
2. Put your CV PDF into `assets/files/cv.pdf`.
3. Replace the placeholder text on `index.html` with your preferred biography.
4. Expand publication entries on `research.html`.
5. Add posts inside the `blog/` folder.

## GitHub Pages

For a simple setup, publish from the `main` branch and the repository root.
