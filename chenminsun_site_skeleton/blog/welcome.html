Welcome to the blog

This sample article shows the structure for future blog posts. You can use this space for research notes, reading summaries, seminar reports, talk abstracts, informal announcements, or polished expository essays.

Since this site is currently implemented as a static HTML/CSS website, each post can simply be an HTML file inside the blog/ folder. Later, if you publish many posts, we can migrate the blog part to Jekyll or another static site generator without changing the overall appearance of the site too much.

A useful first step is to decide what kinds of posts you want here: short announcements, longer notes, reading reports, or lecture-style expositions. Once that is clear, we can create a consistent post template and a cleaner index page.
